# Relayer economics

Relayers stake at least 10,000 $SAKA. Each successful shielded transfer
pays a flat 0.30% fee in the source-chain asset:

- 50% → relayer
- 50% → $SAKA buyback-and-burn

Slashing applies on missed deadlines, invalid proofs, or excessive drop
rates within a 24-hour window.
