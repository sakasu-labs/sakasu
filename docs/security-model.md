# Security model

Privacy depends on (1) circuit soundness, (2) the Solana program correctly
enforcing nullifier uniqueness, (3) the bridge's liveness and finality.

Out of scope:

- User wallet OPSEC. Reveal your view key, reveal your commitments.
- Side-channel attacks against the relayer node.

If you find an issue, email security@sakasu.space before disclosing publicly.
